A pointer that is not initialized to any address is called a null pointer. It is typically initialized to nullptr in modern C++.

int* ptr = nullptr;

Pointers can be incremented or decremented to point to the next or previous memory location of the same type. This is often used with arrays.

int array[] = {1, 2, 3, 4, 5};
    int* ptr = array;
    cout << *ptr << endl;  // Outputs 1
    ptr++;
    cout << *ptr << endl;  // Outputs 2

You can have pointers that point to other pointers.
